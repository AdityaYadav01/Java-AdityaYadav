                                       Basic Linux Commands

1. ls - List down all the files in the current directory.
2. mkdir - To create a new directory/folder.
3. rmdir - To remove the empty directory.
4. rm - To delete all the files and directories
5. rm -r - To force delete the files.
6. ls -a - List down all the hidden files
7. touch - It is used to create a new files. For example: touch 'hello.py'
8. Cat - It is used to display the content of all the files.
9. chmod - To make the files executable. For example: chmod +x "num.py"
10. mv - To move files or to rename a file.
11. hostaname - To know your name in your host or network.
12, locate - To locate a file in linux system. For Example: To locate a file containing words *hello and *this - "locate -i *hello *this".

                                    Text Editor in Linux

1. Vim/Vi or Nano - To open text editor in a linux.
2. vi filename - To create or open file.
3. vi - open a file to name later.
4. vi -r filename - Recover deleted file
5. i/a - To get into insert mode.
6. esc - To get back to command mode.
7. :w - To save changes
8. :wq/ZZ - To save changes and quit text editor
9. :q! - Quit without saving changes
10. dd - To delete a line in command mode
11. 5dd - To delete a multiple lines
12. u - To undo changes
13. ctrl+r - To Redo changes
14. Searching Text in Command mode - /textname
